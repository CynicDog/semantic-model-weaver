# Presentation Template

Use this as the base template for the hackathon submission slides. Modify freely as needed. The goal is to communicate how your solution addresses a business problem using data, and what impact the proposed data application can create.

> Note: Only approved datasets may be used (see participation terms). Using unapproved datasets or failing to cite data sources may result in disqualification.

Both Tech Track and Business Track participants must build on this template.


## Slide 1 — Title

- Team / project name
- Track (Tech / Business)
- Date


## Slide 2 — Submission Checklist

**Tech Track**
1. Submission form (sent individually to registered participants)
2. Project template (this document)
3. Source code zip file, submitted via link
4. Demo video (max 10 min) download link — must include code, model, and app walkthrough; must cover solution, insights, and future strategy

**Business Track**
1. Submission form (sent individually to registered participants)
2. Project template (this document)
3. Project presentation video (max 10 min) — must cover solution, insights, and future strategy

Finals: 10-minute presentation + ~3-minute Q&A. Deadline: **April 12, 2026**.
