# Problem Statement

## Theme

This hackathon is an open-ended challenge — participants are free to define their own problem and solution. The topics below are suggestions only, provided to help spark ideas.

## Suggested Topic Areas

- Demand forecasting for moving-related services (interior, water purifiers, furniture, etc.) and targeted marketing
- Impact of subway proximity on apartment prices in residential complexes
- How population migration (in/out flows) affects real estate prices
- Optimal placement of advertising boards for real estate agencies (TV ad targeting)
- Regional telecom demand forecasting
- Moving season demand analysis
- Rental appliance trend analysis

## Submission Requirements — Tech Track

1. Submission form (distributed individually to registered participants via email)
2. Project template (see TEMPLATE.md)
3. Source code as a zip file, submitted via link
4. Demo video (max 10 minutes), submitted via download link
   - Must include: code walkthrough, predictive model explanation, and application usage guide
   - Must cover: solution & insights, and future action plan/strategy

## Submission Requirements — Business Track

1. Submission form (distributed individually to registered participants via email)
2. Project template (see TEMPLATE.md)
3. Project introduction presentation video (max 10 minutes), submitted via download link
   - Must cover: solution & insights, and future action plan/strategy

## Finals

Finalists present live: 10-minute presentation + ~3-minute Q&A.

## Deadline

**April 12, 2026** — multiple resubmissions allowed before the deadline; only the last submission is evaluated.
